# 📊 Olist Financial Performance Analytics Dashboard

## 🔎 Project Overview

This project presents an end-to-end Financial Operations (FinOps) analytics solution using the Olist E-commerce dataset.

The objective was to clean raw transactional data using Python, load it into PostgreSQL, and build an interactive Power BI dashboard to generate business insights.
---

## 🛠️ Tools & Technologies Used

- Python  
- Pandas  
- NumPy  
- PostgreSQL  
- Power BI  
- Git & GitHub  

---

## 🏗️ Project Workflow

1. Data Cleaning using Python (Pandas, NumPy)  
2. Data Transformation and preprocessing  
3. Loading cleaned data into PostgreSQL  
4. Building interactive Power BI dashboard  
5. Generating business insights  

---

## 📈 Key Business Insights

- Identified monthly revenue trends  
- Analyzed top-performing sellers  
- Evaluated payment method distribution  
- Monitored order status and delivery performance  
- Examined geographic sales distribution  

---

## 🎯 Project Objective

This project demonstrates:

- End-to-end data pipeline development  
- Data cleaning and preprocessing  
- Relational database implementation  
- Business intelligence reporting  
- Version control using Git and GitHub
